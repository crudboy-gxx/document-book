# 第一章 什么是管理
+ 管理实际上是人、物、事三者的辩证关系，不同的组合就会得到不同的结果，而管理，就是确保人与物结合后能够产出最有效的事来。
## 管理的理解
+ 1.管理就是让下属明白什么是最重要的。
  + 企业的管理者常常忙于决定他们认为重要的问题，但是对于下属应该做什么，对于每个岗位要做什么从来不做分析，不做安排，结果每位员工都是凭着对这份工作的理解，凭着自己对企业的热情和责任在工作，出现的工作很难符合标准。
  + 对于评价下属有三个很糟糕的词，悟性、领会、揣摩。
  + 只需要了解两个相邻的上下岗位即可判断企业的决策是否合格。其实管理就是每一层确定下一层级管理这所有明确做的事情。
+ 2.管理不谈对错，只面对事实解决问题。
  + 管理是艺术，同样也是科学，管理的艺术是因为需要面对充满个性的人，科学性是因为管理是有规律可循的。
  + 我们会产生对上司的质疑，最公司规定的质疑，因为我们喜欢用对错来评价管理。但是管理上的对错没有意义，因为管理要解决问题，如果所有的证明你是对的，但是管理结果不好，这样的证明也没有任何价值。
  + 在工作现场发生问题时，管理者往往是分析问题产生的原因，寻找到责任人，解决问题。如果我们用这条管理准则来看，应该是面对事实，解决问题，而不寻找责任。
+ 3.管理是“管事”，而不是“管人”，是界定应该做的事情。
  + “管人理事”是大部分对管理的理解，实际的管理工作中强调对人的管理。但是这个理解是错误的，中国的管理一直处于“人治”的状态，管理行为都是依据对人的判断来进行，而根本的事实是，管理是管事，而不是管人。
  + 事实上，人是无法管理的，从人性的角度来看，每个人都希望被尊重而不是被管理。每个人都本能的认为自己有自我约束的能力，尤其是具有自我实现能力的人，更加觉得提供平台给他发挥比任何事情都重要。所以问题就出现在管理者只关心人们的态度和表现，并没有清晰的界定必须要做的事情，以及做事的标准。
  + 对于大多数员工来说，他们并没有的清晰的指引，应该做什么事情，所以只能凭着兴趣和情绪来做，这样的做事方式，一定是无法评定以及无法控制结果的。
+ 4.衡量管理水平的唯一标准是能够让个人目标与组织目标合二为一。
  + 在管理中，人们都受到一个问题困扰，有能力的员工常常不受组织的目标约束，更为可怕的情况是这些有能力的员工会背离组织。其实，很多管理对忠诚看的很重是因为管理的水平不够，因为老板知道有价的资源有限，所以只能靠无价的情感来弥补。这样做的结果是留下了需要情感满足的员工，对于需要能力发挥得到满足的员工来说只能是离开。
  + 忠诚的衡量应该是员工对于组织的贡献，而非其他。
  + 另一个很普遍的现象是，企业在初创期，所有人都会全力以赴把事情做好，但是到了企业能够存活并有一定成绩的时候，企业开始留不住人。其根本原因是企业不知道如何管理这些员工，更加不知道需要不断关注让人们的个人目标的变化，让组织目标不断得以实现的时候，个人目标也不断地实现并提升。
+ 5.管理就是让一线员工得到并可以使用资源。
  + 除了人事和财务的权利，其他的权利对于管理来说都是次要的。
## 对绩效负责的管理观
+ 1.管理只对绩效负责
  + 企业的绩效包含这效益和效率两个方面的内容
  + 无论采取何种管理形式和管理行为，只要是能够产生绩效的，我们就认为是有效的管理方式和行为。如果无效的就是管理资源的浪费。
  + 关于功劳和苦劳。人们尝尝以苦为乐，认为付出了就是对得起组织，但是我们都很清楚，只有功劳才会产生绩效，苦劳不会产生绩效。
  + 关于能力与态度。如果我们过度关注态度，结果就是导致组织中能干的人干死，不能干的人活的很好。态度的结果导向会导致大家关心态度，而不愿意真正地用能力说话。可是只有能力才会产生绩效，态度必须转化为能力才会产生绩效。
  + 关于才干与品德。品德有态度一样，必须转化为才干，才可以产生绩效。从行为学和经济学方面来看，人本身就是自私、贪婪、懒惰的，这就是管理所面对的人，所以不能用道德来评价一个人。人都会犯错误，所以我们不能够选择品德，管理所有做到的就是让人没有机会犯错。有两种德在先的情况，一个是招聘人员的时候，一个提拔人员的时候，但是我们看到的实际情况确实相反的。
+ 2.管理是一种分配
  + 管理是在责任的基础上所做的行为选择，我们需要在界定责任的同时，配备合适的资源，并让人们可以分享到管理所获得结果。因此，基于责任所做的权利和利益的分配，就是最合适的管理行为。另外，有了要担当的责任，人才才会激发能力和热情，
+ 3.管理始终为经营服务
  + 管理和经营是管理者需要具备的两种能力，经营就是选择正确的事做，管理就是把事情做正确。以往强调的管理不重要，是因为经营是第一位，管理是第二位。经营战略不同，管理的模式也是不同的，经营选择薄利多销，管理上就要选择成本管理和规模管理；经营上选择一分钱一分货，管理上就要做到品质和品牌管理。
  + 管理做什么，必须由经营来决定；管理水平不能够超越经营水平。我们出现问题点就是管理与经营战略的匹配水平上，往往是管理的水平超越了经营水平。
## 管理解决的三个效率
+ 1.使劳动生产率最大化的手段是分工
  + 在泰勒之前，管理是一直存在的，只是没人去了解每个人所做的努力是否有效，也没有分析习惯的做法是否可以改变。泰勒用一生的时间探讨管理的本质问题：管理要解决的就是如何在有限的时间里获取最大限度的产出，也就是使生产率最大化。
  + 获得生产率最大四个原理：科学划分工作元素，员工选择、培训和开发，与员工经常沟通，管理者与员工应有平等的工作和责任范围。
+ 2.使组织效率最大化的手段是专业化水平和等级制度的结合。
  + 关于权利，是个人还是组织的辩证。权利是组织而非个人，组织管理的核心就是让权利从个人身上回归到职位上，也就是组织本身上，只有这种情况管理的效率才会得到。
  + 关于职位和权利。权利是职位一个基本条件，权利需要承载职责，权利不是种种待遇和象征性。如果要发挥团队的作用，我们要让个人权利不再是组织的核心要素，使每个职位的分工与协作成为组织的核心要素。
  + 关于组织结构。国内美的的组织管理状态，是处在职位明确、责任明确、激励明确的组织体系中。
  + 关于组织杠杆。在外部因素快速变化的今天，无法在通过我们习惯运用组织明确的杠杆去做管理调整。
  + 只有贡献了专业化的水平，才算胜任了管理工作。只有职责清晰的分工，权利明确的分配，等级安排合理，组织结构有序，管理的效能才会有效的发挥。
+ 3.使个人效率最大化的手段是创造组织环境，满足需求，挖掘潜力。
  + 管理系统和制度，是需要触发和推动的，虽然公司有人力资源部门，但是员工管理是管理者的自身很重要的工作。
  + 关于激励起不到效果。以往我们的激励是以个人成功为导向的，当个人的成功需要团队来支撑时，原有对于个人激励就无法起到作用。只有让组织的目标和个人目标合二为一的激励就是有效的激励。
  + 关于人才流失。一方面是人才的选择，另一方面没有理解个人在投入产出方面所做出的衡量。
## 如何能管理有效
+ 管理最为重要的作用就是把人联系到一起，共同实现组织的目标。因此管理者承担了重要使命，提升整体的力量，延续个体的价值。
+ 关于传统管理者和有效管理者的区别。传统管理者的特点如下：专注于繁琐的事务，时间是属于别人的；身处什么岗位，就用什么样的思维去看待问题；只专注于事务，忽略了对人的培养。有效管理者的特点：进行时间管理，能确定事情的重要和紧急程度，所有事情都会有序合理的开展；系统思考，从整体和局部去思考，懂得借助整体的力量，局部和个人才会发挥最大的效能；培养人，如果让每个人都胜职责，组织的效率就会提升。
## 企业组织的管理内容
+ 1.计划管理：回答资源与目标是否匹配的问题。
  + 目标是计划管理的基准，计划管理也管理论中被确认为目标管理。资源是计划管理的对象，计划管理事实上是管理资源，而不是管理目标。两者的匹配关系是衡量计划好坏的标准。
+ 2.流程管理：解决人与事是否匹配的问题
  + 解决企业效率的问题，流程关键。流程管理就是人人有事做，事事有人做。
  + 实现流程管理需要改变的三点习惯：打破职能习惯；培养系统思维习惯；形成绩效导向的企业文化。
+ 3.组织管理：回答权利与责任是否匹配的问题
  + 实现组织管理需要两个条件：专业化和分权。
  + 关于专业化。如果组织中以专业化为标准，人们将不在依靠权利和职位传递信息和指令。
+ 4.战略管理：解决企业核心能力的问题
  + 核心竞争力三个基本特征：提供了进入多样化市场的潜能，对顾客重视的价值做出关键贡献，是竞争对手难以模仿的能力。核心竞争力首先应当是深入理解和准确把握市场和顾客需求的能力。
  + 企业必须站到战略的高度从长计议，企业需要审查自己经营的业务、所拥有的资源和能力，观察市场需求和技术演变的发展趋势；识别本企业的核心竞争发展方向，并界定构成企业核心竞争力的技术有哪些，这个战略需要回答的问题。
+ 5.文化管理：解决企业持续经营的问题
  + 企业追求什么样的目标？肩负着何种使命？拥有什么样的价值标准？是企业是否可以持续的根本原因。

# 第二章 什么是组织
+ 组织是为目标存在的，组织里的人与人是不平等的。怎么提高组织能力就成为管理中永恒的主题之一。
## 组织的理解
+ 组织的存在是为了实现目标，组织管理的存在是为了提升效率。
+ 对于组织的理解如下
  + 第一，公司不是一个家。组织不会照顾个人，在组织中我们是用目标、责任、权利来联结的，而不是用感情，这是一个正式的组织。而非正式的组织，使用情感、兴趣、爱好来联结人群的。组织结构的设计从本质上讲就是一个分权、分责的设计。组织存在的理由就是创造价值，如果不创造价值就不可能存在，而创造价值就需要责任、需要权利，从而实现目标，因此感情不是首要的。
  + 第二，组织必须保证一件事是同一组人在承担。关于组织虚设，会导致同一个责任有两组人在承担，同一个权利有两组在使用。同一件事有两个组在做。最终，导致效率低下、人浮于事、责任不清、互相推诿。
  + 第三，在组织中人与人公平而非平等。组织的管理状态，是将自己摆放到组织的结构里面，充分理解组织中自己所处的位置，相应的做出行动和选择，承担自己的责任。
  + 第四，分工是个人和组织联结的根本方法。组织不应该建立很多的管理制度，而应该先解决组织分工的问题，管理制度越少越好，因为制度本身就是一个成本。好公司的状态：一个有机的组织，一个健康有活力的文化，一个有效的分配制度。
## 组织因目标而存在
+ 能支持合理战略的唯一目标就是超强持续的盈利能力。企业需要在不同时期，使得自己的目标明确且单纯，只有这样才能够不至于因为目标的混淆或者多少个目标的选择而耗费了资源。
## 组织内的关系是奉献关系
+ 一个人若不懂得在团队中主动贡献，总是让团队为了他特别费心协调，就算能力再强，也会变成团队进步的阻力。组织内的人与人之间是奉献关系，而不是管理与被管理关系。
+ 奉献关系所产生的的基本现象是：每个处于流程上的人更关心他能够为下一个环节做什么样的贡献；每个部门都关心自己如何调整才能够与其他部门有和谐的接口；下级会关注自己怎样配合才能够为上级提供支持，而上级也会要求自己为下级解决问题并提供帮助。
+ 如何让组织关系变为奉献的关系：
  + 第一，工作评价来源于工作的相关者。目前人员评价的共同点都是以工作结果作为评价的根本对象，如果想要获得奉献的关系，需要改变评价的主体和评价对象。在这个评价体系中，你的上司没有与你构成流程关系，就不需要作为你工作评价的主体。同时，不仅仅评价工作结果，还要评价你的工作贡献。
  + 第二，绝不让雷锋吃亏。在华为基本法里我们看到的条例不是要求，而是企业对每个员工的承诺。“奉献者定当得到合理的回报”。“人力源不断增值的目标优先于财务资本增值的目标”。
  + 第三，激励和宣扬组织的成功而不是个人的成功。在形成每个人的风险行为的时候，需要注重团队荣誉而非个人荣誉的氛围。
+ 不肯付出的人在组织只会做被吩咐好的工作，愿意付出的人就算能力有限，却能带动团队集结众人的力量。一个好组织的每一个成员的第一要事是：主动关心别人的需求。
## 组织处在不确定的商业世界中
+ 如何应对不确定性：
  + 1.管理者需要学会混沌的思维模式。混沌相对的稳定，稳定常常关注的是行动回归到预定的计划上来，而混沌关注不断寻找改进的机会。
  + 2.组织需要构建自己的弹性能力。弹性能力指的是不借助外力，能够自己加压、自我超越的能力。企业进入一个混沌状态的时候，面对的问题都是新问题，没有经验和先例来借鉴，所以组织需要自我超越，自己加压不断改变才是正确的选择。
  + 3.在组织内部打破平衡状态。
  + 4.实现组织学习。组织学习的最根本是要解决组织存在问题的本身，而不是对这些问题产生的后果做出反应。

# 第三章 什么是组织结构
+ 组织结构就是让责任和权利的关系匹配。
## 组织结构是自我约定的关系
+ 公司要设定多少的副职，多少个经理岗位，最重要的是看有什么责任需要分配，这里没有臃肿的问题，最重要的是界定责任。
## 组织结构的功效
+ 1.组织结构所要解决的是权利与责任是否匹配的问题。
  + 组织结构需要清晰的设计出沟通线、控制线、责任线和权力线。责任和权力是纵向安排，沟通和控制是横向安排。
  + 纵向安排的两个问题，一个是设计多少层级，另一个是公司主业务线是什么。层级设计的原则是以考核点为准。业务线是以主营业务为标准。
  + 横向安排考虑的问题是：需要多少个职能部门完成资源的专业安排。最关键的是尽可能地减少细分，突出关键职能就可以，部门越少越好。
+ 2.组织结构更要依据责任而不是权利来设定。
  + 企业在组织结构设计上常见的错误：第一种，面向董事长，屁股对着顾客的结构。第二种，条块结构，各自为政。
+ 3.组织结构可以重新建立组织和个人之间的心理契约。契约为员工与组织相互之间期望的总和。
  + 如何保持契约的良性循环：第一，建立开诚布公的沟通体系，让员工知道自己在结构中的位置。第二，确保结构的形成是公平的。不公平性会造成员工的负面反应。第三，恪守承诺，心里契约的构成基础是信任，为了稳定现有员工的心理预期而做出的承诺，可能成为未来食言的证据。
## 组织结构设计原则
+ 第一，指挥统一，一个人只能有一个直接上司。
+ 第二，控制幅度，一般的管理跨度比较合适的是五六个人，越到基层跨度越大，越到高层跨度要越小。
+ 第三，分工，分工有横向和纵向两个方向，纵向分工是经营分工，这条线决定绩效的分配、权利分配，纵线又被称为职权线。这条线上不保证承担绩效的人权利最大，而不是职位越高的人权利最大。横向是资源线，公司所有的资源都在这条线上进行专业分配，保障业务部门能够获得支持。横向线需要的是专业化，而不是细分。
+ 第四，部门化。必须把同一件事的人放到用一个部门，由一个经理来协调。
## 组织结构需要配合企业发展的需要
+ 关于影响组织调整的因素。策略、规模、环境、技术。如果这四个方面没有发生变化，组织结构可以不改变。
+ 从创业、成长、发展、持续发展四个阶段阐述组织结构的状态。目前大多数在第二阶段，而此阶段最重要的是构建专业队伍，而不是招聘经理人。
+ 一个企业的发展，一方面需要企业文化，另一方面需要契合的客户，同样重要的需要一个可以分项的结构。
## 组织结构的特殊效能
+ 组织结构如何让人获得满足感和工作成效:
  + 第一个层面是职权阶层（管理层）。管理层会获得比别人很多的资讯和决策机会，他们可以掌握和运用资源。
  + 第二个层面专业化岗位梯队建设。必须为专业化设计梯队，为优秀员工提升晋升空间，而不是挤管理岗位。
  + 第三个层面是部门的划分。部门的划分可以按照目的划分，也可以按照流程划分，无论哪种划分，在明确的部门里面，成员最具有这个部门专业领域的权威性。
  + 第四个层面是授权和分权。组织职能是一种分权，领导职能是一种授权，授予的权利依旧在领导手里。
  + 第五个层面是形式化的程度。称呼和头衔的形式化。
  + 第六个层面是控制幅度。
  + 第七个层面是专业化。

# 第四章 什么是领导
## 领导的理解
+ 领导是指影响别人，以达到群里目标的过程。领导者负有指导、协调群体活动的责任的人。领导的特性是，作为管理职能，领导借助于影响力发挥作用而非职位。影响力有两部分构成，权利和魅力。
  + 1.权利。权利产生影响力，可以从五个角度来体现：法定权、专家权、奖赏权、惩罚权、统治权。
  + 2.魅力。外貌、类似性、好感回报、知识、能力。
## 领导者和管理者
+ 领导者和管理者存在一致的岗位职责。领导者的侧重点是定方向、构建团队、促进变革，管理者侧重于解决问题、稳定团队、按章行事。领导者承担的责任是让企业有一个明确的方向，不断适应变化，有一个核心的管理团队，确保组织的成长。管理者只对绩效负责，解决问题，保持稳定，制度和规范得以贯彻和执行。
+ 中国企业中，领导者常常关注绩效和当前，而管理者关注成长和未来。表现出来的现象就是，中层管理者总是谈论公司战略和变革，而不谈论制度和规范的执行和面临问题的解决方案和行动；领导者也关注战略，但是更多关注绩效和执行力，领导者谈论的最多的问题就是管理者的问题。其实管理不需要换位思考，而是需要做好本职工作。
+ 从一个组织中来看，领导只有一个人，绝大多数是管理者。但是相对的，在小的组织中，管理者又承担了领导者的职责，所有大多数人需要平衡两个角色。所以需要具备实现绩效的能力，但是内部促进变革、形成好的管理团队、不断让部门方向跟公司的方向一致。
## 领导的技能
+ 领导需要的三种技能：
  + 人际技能是指和人们一起工作的能力，包括人际关系技能、组织技能、解决冲突技能。
  + 概念性技能是指明白整个机构的复杂性及本人的工作适合于机构内任何关节的能力，包括信息处理技能与制定决策的技能。
  + 技术性技能是指执行任务而必须的知识、方法、技能。
+ 这三种能力，对于不同层级的管理者侧重点不同。人机技能是否需要的，对于基层管理者技术性技能更重要，要能够直接解决问题。对于高层管理者概念性技能更重要，要能够复杂问题简达化。概念性技能对于一个人的上升，有决定性作用。
## 经典领导理论的应用理解
### 应该适应领导者的风格而非改变他
+ 菲德勒提出了领导方式取决于化境条件的著名论断，可概括为：当环境条件有利或不利的情况下，工作导向型的领导容易取得成效；当环境条件处于中等有利情况，员工导向型容易取得成效。
  + 影响环境的因素包括三部分，第一领导者与成员的关系，第二职位权利，第三任务的具体化。
### 没有不好的士兵，只有不好的将军
+ “赫－布理论”，我们要根据员工任务成熟度来划分员工，在根据这个划分选择不同的领导风格。任务成熟度只的指下属从是某一任务的能力与动机。
+ 员工成熟度与领导风格：有心有力-授权型，有心无力-参与性，无心有力-推销型，无心无力-吩咐型。
+ 不管是什么样的员工，工作的绩效取决于管理者的水平，而不是员工的水平。员工可以是不同状态的，甚至是无心无力的，但是管理者具有不同的水平来应对，无心无力的员工也能取得好的工作绩效。企业一定要关心管理团队的创造和培养，只要他们有水平，所有员工就会发挥作用。
## 找到途径满足需求，目标就会达成
+ “途径-目标理论”，在豪斯眼里，领导者的基本任务就是发挥部下的作用，而要发挥部下的作用，就要帮助部下设定目标，把握目标的价值，支持并帮助部下实现目标，在实现目标的过程中提升部下的能力，使部下得到满足。
+ 在路径目标理论中，领导的风格不在重要，最重要的是找到实现目标的路径，让员工得到工作的绩效满足感和工作成绩。领导行为可以分为四种，第一种是指导型行为，第二种是支持型行为，第三种是参与型行为，第四种是成就导向性行为。行为的选择主要考虑下属的特征和环境。
+ 管理者主要作用就是为下属提供支持和帮助，消除实现目标过程中的障碍。所以管理者具体任务就是识别每一位下属的个人目标，知道他们的需求和愿望，了解他们的能力，同时知道他们所承担的工作任务和特征，建立薪酬体系，使个人目标与组织绩效沟通，已获得满意的绩效水平。
### 选择的途径和员工需求以及承担的任务特征要保持一致
+ 因为员工需求特征越来越成为重要的影响因素，管理者需要做的就是找到员工的需求特征。
+ 根据不用的需求特征与任务，要选择不同的途径。对于年轻人，需要明确告诉他什么该做，没什么不该做。
### 对有能力的员工需要尊重和授权
## 对职业经理人的管理方式
+ 一个是例外管理，一个是根据业绩给与合理的报酬。
+ 物质报酬四个特征：老板和职业经理人达成共识，物质能达到经理人预期，报酬要跟经理人绩效挂钩，老板满足职业经理人切身利益。
+ 关于例外管理，是指超越绩效目标的情况，就需要干预。绩效目标一个是业绩目标，一个是费用预算。在目标之内，职业经理人可以按照自己的设计工作，会得到充分的授权和信任。一旦超出就需要管理。
+ 对经理人的管理遇到的困难，一个是物质方面，出现不兑现的情况；在例外管理方面，因为没有合理的预算和明确的目标，也无法取得较好的效果。
## 核心人才的管理方式
+ 1.发挥领袖的影响力。面对核心人才，是需要通过价值观达成与使命的认同，而不是上下级关系的认同。这就需要领导这能够沟通使命和价值观，而不是仅仅沟通工作内容。因为核心人才，在专业能力和管理能力上他比你强，而且天天做事情，你的意见或建议不见得对他有帮助。
+ 2.真正的个人关心。对于和核心人才需要关注到他们的个人需求和成长，必须是以独立的、个体的认知来处理与核心人才的关系。很多管理者没有真切的对下属的个体认识，对于组织的标准和目标可以清晰地理解，但是对于个人的标准和目标理解的就不够。企业的人力资源只关注绩效和个人行为的关系，而不关注绩效与个人目标之间的关系。如果被称之为核心人才，这些员工会具有自我实现目标的能力，也就会具有多种需求，而非单一、共性需求。
+ 3.心智的激励。人的心智决定了行为的选择和决策前的逻辑判断。中国在心智上，有两方面做存在先天缺陷。第一个是身边的人比自己好的时候，如果我们无法接受身边的人比自己好，我们也就会失去与他们合作的机会。第二个是枪打出头鸟。当一个人特别优秀的时候，他身边的人不是聚在一起商量如何向他学习，而是商量如何让他回到大家身边。
## 如何让授权有效
+ 为什么要授权。授权可以腾出自己的时间做你想做的事情，授权可以让下属成长起来，授权可以充分发挥人的积极性。授权最大的作用是培养人，只有承担了责任，人才会成长，而不是仅仅提供了岗位。
+ 如果下属的成熟度不够，而又做了充分的授权，结果就是无法达成目标。其实问题的关键不在于下属是否成熟，而在于我们如何授权。
+ 在授权中，资源的运用、方法的选择、实现手段的安排都可以授权，而目标的设定不能授权。如果将目标的设定也授权出去，就容易造成目标无法实现的情况。这样授权的时候，目标就无法成为组织管理的目标，而是下属和组织寻求资源的理由。
+ 授权需要注意：第一，机构越大越要授权；第二，任务和决策越重要，越不能授权；第三。任务越复杂越授权；部属之前互相不信任，不能授权；第五，部属责任心不够，不能授权。
## 利用环境
+ 对领导者而言，最重要的并不是你自己的本人拥有什么样的天赋、能力，而是最重要的是能够利用环境。因为自己再有能力，如果不能够集合智慧，顺势而为，也不可能做出成就。所以，要有能力营造有利于你自己的环境。
+ 新官上任最重要的融入环境、认识环境并利用环境，而不是来改造环境。环境其实无法改造，只能利用。就算要改革，也必须获得原有环境中一部分人的同意，才能取得效果。目前的关键，不是判断原有的环境是否对错，而是获得管理的成效，对于管理成效而言，利用环境可以达成，改造环境需要付出额外的毅力和代价，同时改造所带来的不稳定成本极其高。
## 向上管理
+ 很多人都认为他必须对上司负责，而同时需要管理好下属。这样的管理逻辑是非常普遍，但是恰恰是有问题的。
### 管理自己的老板
+ 管理工作需要资源，而资源的分配权在你的上司手上，你所需要的就是获取资源，这样你就需要对上司尽心管理。向上管理的核心就是有意识的配合上司取得工作成效。
+ 建立培养良好的工作关系：和谐的工作方式，能够采用双方接受的形式处理问题、交流看法并明确各自的职责。相互期盼，通过提升期望来提升各自的能力。信息流通，保证你与上司之间的信息流通，且不能借助第三者来流通，更加不要有所保留。诚实可靠，向上管理是一种相互信赖的关系。合理利用时间和资源，时间最好的作用是能够带来机会，而上司的资源最直接的功效就是为你的工作提供帮助。迎合上司的长处，尽量避免上司的短处。
### 技巧和注意的问题
+ 1、利用上司的资源和时间。有时候觉得他的变化接受不了，因为与他的沟通少，不知道他是怎么想的，所以不能保留与上司的沟通时间。
+ 2、保持正式的沟通。要在工作时间内探讨问题，交换意见，获得指令以及指引，甚至是获得帮助和支持。
+ 3、发挥上司的长处。下属要了解上司的长处是什么，懂得如何配合上司。
+ 4、欣赏与信任。

# 第五章 什么是激励
+ 激励就是让人们自己做出选，并愿意付出。
+ 激励的两个角度：第一个角度是激励一定想办法让工作变成游戏。第二个角度是激励要站到对方的角度来做，不能从自己的角度去做。
## 人为什么要工作
+ 因为工作理由的丰富性，激励是一个复杂而困难的工作。如果要进行归纳，大致分为五类：赚钱、消耗能量、社会交往、成就感、社会地位。
## 涨工资并不会带来满足感
+ 双因素理论，激励-保健因素理论。把提供给人们的工作条件细分为激励因素和保健因素。所谓保健因素，就是一个人开展工作所必须的条件，如工资、岗位、培训、福利等；所谓激励因素，就是一个人做好工作所需要的条件，如晋升、奖金、价值的肯定、额外的工作条件等。
+ 保健因素不会有激励作用，当缺乏时，人们会不满，当存在时，人们的不满只是减少，但是不会带来满足感。
+ 如果使用保健因素，就要让绝大多数人的得到，这样才能让不满的人减少，否则得到的没有满足感，只是降低了不满，得不到的员工会非常不满。保健因素只能升，不能降。如果使用激励因素，就要确保获得激励因素的员工是很少的一部分人。如果激励的人数过多，激励就会降为保健因素。激励因素必须是可以变动的，不能固定，一旦固定下来又要变为保健因素。
+ 企业在做人力资源的薪酬设计的时候，要参照同行业的水平，也要参照企业的经营规模和水平，对于同行水平对应下，不要好高骛远。否则你用很高的薪水，挖了强大的人来，而给他的业绩指标又不是这么强大，最终会毁了这个人，也毁了公司。
## 最低层次的需求如果得不到满足，影响最大
+ 马斯洛需求理论，生理需求、安全需求、交往需求、尊重需求、自我实现需求。对于马斯洛需求理论而言，第一，人的需求是由低向高递进的；第二，最低层次的需求没被满足最有影响力；第三，以满足的需求不再有激励效果。
## 人不流动也许是因为安于现状不求发展
+ ERG理论，生存、联系、成长论。对于ERG理论而言，第一，人的需求不是由低向高递进的，而是多种需求同时存在的。第二，当一个人的需求满足收到挫折时，这个人会选择降低自己的需求，放弃更高的需求。当成长需求无法得到满足时，这个人会选择回归到联系的需求中，如果联系的需求也无法满足，他会选择回归到生存的需求中。第三，确定人们需求的影响因素是，他自己的发展水平和他在团队中的经验。
+ 管理人员可能认为人力资源稳定是一个好事，但是从ERG理论出发，我们所关心的不是人员流动性，而应该关心人们不流动，留在组织的关键因素又是什么？如果大家都不流动，不是因为需要发展，而是因为生存需求被满足，这样的组织一定得不到发展。同样的，我们也要关注那些满足感很高的员工，应该了解他们的满足感来源于什么。如果员工的满足感不是来源于成长的话，对组织也是有害的，因为组织不可能获得成长。
+ 在组织中，一个人的发展，取决于他需求的强度，但是这个人的需求强度又取决于他自己的发展水平和他在团队中的经验和位置。
## 不要满足需求而是引导需求
+ 工作中有三种重要的动机：成就需求，权利需求，亲和需求。
+ 具有强烈的成就需求的人渴望将事情做的更完美，提高工作效率，获得更大的成功。他们希望得到有关绩效的及时明确的反馈信息，从而了解自己是否有进步；他们设定具有挑战性的目标，不喜欢凭运气获得成功，不喜欢接受在他们看来特别容易或特别困难的任务。
+ 关于需求与工作绩效的关系：喜欢独立负责、可以获得信息反馈和中度冒险的工作环境，会从这种环境中获得高度激励。高成绩者也有相应的缺点，那就是只对自己的工作绩效感兴趣，并不关心如何影响人去做好工作。但是，最优秀的管理者往往是权利需求高而亲和需要很低的人。
+ 员工成就的大小，取决于他们自我成就的激励或外部成就激励。如果完全去满足员工的需求，这是不可能的事情，所以不要在满足员工需求方面花太多的脑筋，而是应该选择能够自我激励的员工，选择成就动机搞的员工。我们要不断激发员工的成就需求，而不是去满足员工的需求。
+ 关于麦克利兰的成就理论理论，有两点需要了解，第一人们是需要外部激励的，不要认为每个人会自己找事情做，会以很高的标准来要求自己。第二要想尽一切办法去激发人们内在的成就欲望，因为志友激发内在的欲望，人们的成就才能够真正获得。
## 满足感不一定带来高绩效
+ 满足感是个人需求获得满足而引发的，但是个人需求的满足和工作绩效并没有直接关联。很多时候人们个人需求是非常个性化的，和工作没有任何的联系。
+ 面对这种情况，我们要调整：第一要满足员工的需求，让员工货的满足感。第二让员工的满足感来源于工作本身而不是个人需求。工作本身可以用五个指标来说明：薪资、晋升、信任、同事关系、工作本身。薪资和晋升是人们满足感两个非常显而易见的指标；信任也很重要，因为新人就会减少紧张程度，不需要态度的监督和猜疑；在一个同事关系融洽的环境中，同事之间会互相帮助，工作的推进也可以很好的开展和取得成绩。
## 激励不发挥作用的情况
+ 第一，工作超量所导致的疲惫。当一个人工作能力很强的时候，就会承担非常重的工作量，当然这会相应获得高的肯定。当工作超量之后，会引发疲惫，导致无法热爱这项工作，他就会离开。
+ 第二，角色不清，任务冲突。清晰的职责和分工，是员工获得工作绩效的前提。无法获得清晰分工的现象同样存在，甚至角色不清，他们不知道直接汇报路线是什么，而已不清楚什么样的按工作标准可以做参考。他们承担这多种任务、多种角色。
+ 第三，不公平的待遇。因为在人们的心中，只有公平存在，所在的考核和奖励才会真正有效。如果公平本身不存在了，那么考核和奖励只是形式上的，而不是真正意义上的。
+ 如果出现如上三种情况，就不要从激励角度努力，因为这不是钱和承诺能起作用的。这种情况，就不要动用激励的措施，而是切实的改变人们所处的工作状态，合理的工作量设计、清晰的职责、明确的任务以及公平的待遇。
## 不公平是绝对的
+ 在管理状态下，因为分工不同，承担的责任不同，所获得收益也是不同的，因此不公平是绝对的。因为公平是一种感觉，是一个人的判断，只要我们能够合理的提供判断标准，公平感就会出现。
+ 公平感源于：我获得/我付出=他人获得/他人付出。当人们比较的是付出，而不是获得的时候，就会产生公平感。公平理论的核心就是通过比较每一个人的付出，使人们获得公平的感觉。
+ 在运用公平理论中，最大的困难是管理者认为公平的东西，员工认为不公平。这非常普遍，因为所处的位置不同，承担的责任不用，看问题的角度不同，对公平看法差异很大，所以要了解员工的真实想法。第二个问题是实际人们并没有对公平看的那么重，人们在意的实际上是平均，要注意“不患贫，而患不均”的想法。
## 人会成为他所期望的样子
+ 关于期望理论，个人行为倾向的强度取决于个体对这种行为带来结果的期望程度，以及这种行为对个体的吸引力。激励力量=目标价值*期望值。一个期望对于个人影响是的实验。
+ 期望理论运用的三个基本条件：第一期望价，也就是设定目标，必须让成员相信这个绩效目标是可以实现的；第二是媒介，媒介是需要有获得信任的载体和措施；第三是对期望目标的评估，确信这个目标的价值。简单讲，期望理论其实就是设计一个绩效目标，并让人们确信这个绩效目标，最终实现这个绩效目标。期望理论是培养年轻员工的主要方法，因为年轻人都对自己有很高的期望，因为可塑性强，完全按照我们的期望来塑造他。
+ 运用好期望理论需要做到几点：第一设计的目标不要太高，一个没有人相信会实现的目标，是一定不会有激励效用的；第二需要有媒介，需要设计一个让所有人认为公平可信的载体；第三承诺要兑现，不管多少人达成期望，都一定要兑现承诺。
## 金钱是最重要的激励措施
+ 大多数人的理解：对于高层次员工来说，金钱对他们已经不重要；对于低层次的员工来说，没有钱万万不能，钱又不是万能的；自我实现和获得成就比金钱更重要。这样的理解是对金钱的看法，而不是从激励角度来看的，在激励上，金钱非常重要。
+ 衡量采用激励措施是否有效，有三个基本特征：重要性、可见度、公平感。金钱只是激励的一种措施，按照重要性和可见度来衡量，还有晋升、福利、社会地位等。
+ 激励措施最重要的是满足员工的需求，理论上是没什么错误，但是事实上如果以员工需求来安排激励的话。就发现很困难，一方面是每个员工都不用，另外也很难了解到真实的想法。
## 成本最低而且最有效的四种激励措施
+ 鼓掌、赞美、鲜花、隆重的仪式

# 第六章 决策如何有效
+ 集体决策，个人负责。
+ 一个最重要的领导行为，就是怎么快速决策，并保证这个决策是最有效的。决策是领导者的日常管理行为，同时决策本身又和很多问题先关。
+ 从决策的分类来讲，它可能跟组织有关，也可能和个人有关，可能是例行的， 也可能是非例行的，可能是预料中的，有而可能是不可预料的。而决策会决定选择。
## 决策的目的是为了执行
+ 决策是为了能够执行，而不是追求正确性。决策的正确性不是决策本身，而事决策得到执行的结果。其实对于决策来说，主要是看决策者做出决策的时候，能不能让决策执行到位，而且是否可以坚持到获得决策结果。
```
1识别问题、2确定标准、3分配权重、4拟定方案、
5分析方案、6选择方案、7执行方案、8评价方案。
```
+ 从步骤1到6的过程是决策过程，决策本身就是选择，但是如果我们把决策落地执行的话，就要解决问题，而不是做出选择。所以决策是要解决问题而不是简单做出选择，只有把问题解决了，决策才会获得结果并被检验。
+ 选择决策和解决问题的区别：如果我们确保决策是可以执行到位的，那么执行决策的人开始就要参与决策。往往造成我们决策无法获得实施的问题，就是做出决策选择的是一组人，执行决策的是另外一组人。
+ 决策的第一个重点，不是判断这件事情要不要做，而是判断能不能找到人去做。并且，做出决策之前，要先确定谁来执行这个决策，之后才开始展开决策的过程。很多情况是，先决策，然后确定谁去做。
## 重大决策必须是理性决策
+ 管理决策困难的地方是，既要面对人，也要面对事。所以管理者要具备自然科学的思维方式，也要具备社会科学的思维方式。自然科学的方式就是数学的方式，不断的细分或者趋近事实，用实验不断接近真理的方式来获取对于事务的判断，这种允许在实验中犯错，最后获得成功。社会科学无法使用实验的方式，和自然科学正好相反。
+ 因为这两种特性，因此进行管理决策的时候，不能简单依据数据，也不能简单的凭借经验来做判断，对于重大决策，必须考量诸多的条件和因素，才能做出决策。一个是要以实时、以数据、以真理说话的自然科学的态度，一个是以本质、以任性的社会科学的态度。管理上的任何新的变化，都不要全面展开，要经过局部试点实验，获得成功的经验，之后在全面实施。
+ 理性的决策怎么做：
  + 1、识别问题。问题识别要真实的考虑现实和理想之间的差距。切勿让专家做决策，一个是他们都是只基于理想做分析，第二是专家依据的数据都是整理过的，第三是专家并不需要对决策承担责任。
  + 2、确定标准。我们必须清楚的知道什么因素和决策相关，这些因素必须是可观察、具体、可测量。
  + 3、分配权重。依据相关因素的重要性，对标准进行权重分配。
  + 4、拟定方案。列出能成功解决问题提的可行方案，越多越好。
  + 5、分析方案。分析方案的可行性、代价、风险等。两点注意点，第一分析方案应着重于建议，而不是人，分析要依据方案本身，不能因为方案是领导或者专家提出而重视。第二分析社会愿望因素，还是取悦上司还是同事。
  + 6、选择方案。是否是妥协结果，是否所有决策人满意，是否受权利影响。
  + 7、执行方案。接受程度和资源限制。
  + 8、评价方案。
## 集体决策 个人负责
+ 很多时候，决策是一个人确定，但是要集体来讨论和承担，而且常常借助于集体的力量来体现个人意识，这种是错误的。
### 集体决策
+ 集体决策因为集合了群里的智慧，相互碰撞和迁就，形成结果。它不是最好的结果，但是一个折中的选择，同时也是风险比较小的决策。
+ 做到集体决策至少需要满足一些条件：相对少的人数且相互信任，互补的技能，共同责任感基础上的具体目标和共同的工作方法。
  + 是否可以短时高效的举行会议；是否能够高频度无障碍的交流沟通；是否能够开诚布公相互配合做事；是否能了解其他人的作用和技能。
  + 技术性或职能性的技能；解决问题和做决策的技能；人际关系的技能。
### 个人负责
+ 集体决策的实现需要个人负责来保证，没有明确的责任意识，就无法让管理变得有效，就无法承担经营的后果，就没有风险控制。
+ 实现个人决策，需要确定这样一些要素：
  + 每个人是否愿意为集体的目标和工作产出负责任。
  + 团队成员是否干到对衡量指标负有责任。
  + 是否明确什么是团队个人的责任，什么是大家共同应用的责任。
## 个人决策的局限性
+ 对于管理者而言，决策是她必须做的选择，因此在管理者决策的过程中，自己有很多局限性，这些局限性是就是决策的障碍。
### 四个“人际错觉”
+ 1、首因效应。人们往往会看中一个人的第一印象。
+ 2、晕轮效应。人们会以面概点，人们会被外在的东西所蒙蔽，而且依据这个现象去判断。尤其是面对加班和不加班的两个同事的时候。
+ 3、新近效应。在做决策的时候，最新近发生的事情，会起决定作用。尤其是面对绩效考核的时候。
+ 4、角色固着。职业的角色、身份的角色等都会影响人们的决策。
### 不易察觉的偏好
+ 人们会不自觉的察觉偏好，例如招聘是总喜欢选择相同认知的人，或某一个地方的人，或某一个学校毕业的学生。我们在做决策的时候，不管怎么要求决策者理性，我们的承认是用自己的标准和概念来做出判断的。
## 快速而有效决策的五种方法
+ 在决策中，最重要的就是快速决策以保证效率。我们需要知道在何种情况下，采用何种决策方式比较合适，同时了解决策的关键点，以此来掌握决策的方法。
+ 决策五种方法：独断式决策、咨询式决策、群体决策（多数人控制）、群体决策（共识）、授权。
+ 决策有效的评估标准：第一，决策方案的品质和合理性；第二，成员的接受和支持程度。这两个标准的侧重点不同，采取的决策方式也就不同。
## 群体决策不是最好的决策
+ 群体决策最大的好处是控制风险，而不是得出最佳决策，因为群体决策是一个折中、考虑多方面因素的选择。大企业要支付系统和流程的成本，来实现控制风险。
+ 一般而言，小企业的主要是抓住机会，通常会一个人决策；而大企业，因为决策者离现实又远了，信息会变少，还是一个人决策会带来风险。大企业的盈利已经不是最主要，控制风险才是最主要的。
+ 影响群里决策的几个关键因素：第一参与群体决策的人数不要太多。第二每个人都不必须认真负责全程投入。第三成员背景一定要不一致，责任要不同。第四决策中避免出现虚假的相应。第五要充分的让所有人表达意见。第六不要在意流程而要在意责任。

# 第七章 什么是计划
+ 管理只有两种基本模式，一个是绩效管理，一个是计划管理。
+ 计划管理是觉得目标与资源是否匹配的问题。企业的管理活动中，最基本的就是目标和资源，人和事，权利和责任。
+ 企业管理中如何兼顾存在长期与短期、如何既要变化又要稳定、如何平衡效率与效益。
+ 计划中的5W1H，做什么？为什么做？何时做？何地做？谁去做？怎么做？
## 目标是不合理的
+ 目标的第一特性：目标是对未来的预测，预测无法合理，所以目标一定是不合理的。因为目标是觉得未来的问题，而不是现在的问题。设定目标的时候并不是看企业有什么样的资源和能力，更重要的判断发展的趋势以及面对的竞争。如果企业基于自身的能力和资源来定，也许目标合理，能够实现，但是目标实现的时候企业已经被同行淘汰。所以对于目标不是探讨合理性，而是探讨必要性。
+ 第二特性：必须保证行动的合理性。计划的本质上讲是寻求资源的计划，不断寻求资源来实现目标。所以有两个需要注意的问题，一个不要和上司探讨目标的合理性问题，另一个是与上司探讨资源的问题。好的管理者一定是承接目标，但是寻求资源，只有主动承担目标而不断寻求资源的人，才能体现经理人的价值。
+ 目标并不是关键，关键是实现目标的行动，也就是寻求资源的行动要合理。
## 一定要在意行动
+ 很多情况，管理者都在分解目标，检查目标实现，做季度和年度目标的实现。表面上看没什么错误，但是对于目标实现没有太多帮助。如果我们时间都花费到讨论、分析和总结实现目标行动的合理性、资源的安排、时间控制，目标达成率一定会更高。
+ 人们犯的最大错误，是把计划等同于目标分解，所以管理者认为只要把目标分解了，计划工作就完成了。另一个是年底做计划的时候，大家任务这是一个上交的东西，而不是一个管理的职能，随机的设计，计划到年底才拿出来对比。
+ 只要把实现目标所要做的事情设计出来，让人们清楚自己要承担的责任，知道时间上要去，以及展开计划的点，计划就可以指导人们的工作。
## 计划管理如何推动企业发展
+ 计划管理者对高、中、基三层管理这的目标是不同的。高层对长期和变化负责，公司有没有未来取决于他们；中层对稳定和效率负责，公司是否有效率，是否有合适的人才队伍，取决于中层；基层对短期和效益负责，公司是否具有盈利能力，是否可以降低成本、保证质量，取决于基层。
+ 而很多企业无法平衡长期与短期、变化与稳定、效率和效益，主要原因是让高层负担所有责任，无论是成本问题、质量问题、盈利问题，还是人力资源问题、效率问题，并没有划分不同管理者承担不用的责任和目标。
+ 我们犯了一个及其大的错误但是却不自知，我们给高层管理很高的待遇和权利，但是他们却做着中层管理者，甚至是基层管理者的事情，不断的为成本、质量和效率花费精力，他们没有促进变化、关注投资汇报以及企业的未来。
## 制定计划的关键
+ 制定计划的关键是缩小实现和理想的差距，只要行动可以缩小差距，这个行动就是合理的。所以计划最关键的就是找到解决差距的策略点，围绕着这个策略点展开资源和行动，目标就会实现。
+ 大多数人制定计划的时候，并没有关注到差距的问题，只是分解目标，探讨目标的可行性，而没有了解理想和现实之间的差距，更没有根据差距来确定行动方向和资源获取。
## 计划的有效性
+ 如果公司是让下面的部门做计划，在汇总到公司，之后在确定计划，这样计划的有效性就大打折扣。因为人们的计划的目的不同，如果是为了完成考核，计划目标就会很小。
+ 如果保证计划有效性，需要从公司层面确定计划目标以及资源安排。计划一定是财务部门和计划部门一起统筹来做。计划最核心的部门就是预算，确定了资源的多少，然后在安排目标和资源的分解。计划有效性的第二个关键是激励政策的安排，激励要随着计划确定下来，而不是先确定目标责任，之后在公布奖励办法。因为激励政策决定了人们对这个计划的呈弱程度。
+ 计划执行的三个重要要素：第一管理人员对计划的态度，正式的签订目标责任书会给管理人员一种认识，对待计划要认真，要遵从；第二不要用原来的方法解决问题，计划失效的原因是计划和环境冲突；第三上司的支持不够充分，计划的实现需要资源，只有上司才能解决资源的分配问题。
## 目标管理
+ 彼得德鲁克指出，管理的成效取决于目标的设置和目标协调。通过目标设置激发出动力，积极的调动是重视目标和追求目标的过程。
+ 计划的实现是依据目标管理来进行的，其中包括目标设置和目标管理。目标设置既要有一定难度又切实可行，要遵循SMART原则，具体的、可衡量的、可到到的、相关的有时限的。
+ 目标有两种，一种是经营目标，是财务上的销售额、利润、成本、质量等。另一种是管理性目标，是效率、流程、服务等。
+ 目标管理是让员工亲自参与工作目标的制定，在工作中实行自我控制，并努力完成工作目标的一种制度。目标管理的注意事项，第一必须设定总目标，总目标要与分目标方向一致；第二每一个职工的分目标就是企业总目标对他的要求，也是他对企业总目标的贡献，并依据这个目标对他进行考核。第三承认每个职工有自我成就、施展才能和希望自治的需求。第四为巩固成绩，必须注意人的行为，并给予奖励。
+ 目标管理要自上而下的分解目标，自下而上得层层保证。目标管理出现问题，主要出现在措施具体化和细分化方面做的不好，更没有层层保证。所以目标管理的核心是实现目标的措施具体化，而不是目标的分解具体化。目标设定后要对目标进行沟通，花时间与每个人讨论实现目标的措施，只有把措施讨论清楚，目标管理才能到位。
## 为什么“计划没有变化快”
+ 计划没有变化快的原因是计划本身没有设计好，没有将变化包含在内。计划需要有前瞻性和柔性，为了适应环境的不确定性，战略的柔性成为人们的思考方式。
# 第八章 什么是控制
